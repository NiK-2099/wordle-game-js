# wordle-game-js
# 🎯 6-Letter Wordle Clone

> A 6-letter Wordle clone built using JavaScript.

---

## 📌 Overview

This project is a browser-based word guessing game inspired by Wordle.
Players have a limited number of attempts to guess a hidden **6-letter word**, with feedback provided after each guess.

---

## 🚀 Features

* 🔤 6-letter word gameplay
* 🎮 Interactive UI
* 🟩🟨⬜ Color-coded feedback system
* ⌨️ Keyboard input support
* 🔁 Multiple attempts per game
* ⚡ Fast and lightweight (pure JavaScript)

---

## 🛠️ Built With

* HTML
* CSS
* JavaScript

---

## 🎮 How to Play

1. Enter a **6-letter word**
2. Press **Enter** to submit your guess
3. Get feedback:

   * 🟩 Correct letter in correct position
   * 🟨 Correct letter in wrong position
   * ⬜ Letter not in the word
4. Keep guessing until you solve it or run out of attempts
